Starting a Blog

12 December 2014 on Getting Started, Ghost, Docker

This blog is powered by Ghost via docker and hosted as static files using Github pages.

Settings this up took just a few steps.

  1. Start Ghost

     docker pull dockerfile/ghost
 docker run -d -p 80:2368 -v <override-dir>:/ghost-override dockerfile/ghost

  2. Setup Ghost at http://localhost/ghost

  3. Set blog URL via config.js and restart ghost

  4. Convert the Ghost blog into static files using buster

    buster setup
buster generate --domain <localhost:80>

  5. Push files to github

    git push
